# Loan-approval-prediction
End-to-end Machine Learning project for Loan Approval Prediction using Decision Tree and Random Forest with data preprocessing, EDA, feature engineering, and model evaluation.
## Overview
This project predicts whether a loan application will be approved based on applicant details such as income, credit score, loan amount, employment history, and city. It demonstrates a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature encoding, model training, and performance evaluation.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Decision Tree & Random Forest models
- Model evaluation using Accuracy, Confusion Matrix, and Classification Report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Conclusion
This project showcases an end-to-end classification pipeline for loan approval prediction. Among the implemented models, Random Forest achieved better performance, demonstrating the effectiveness of ensemble learning for classification tasks.
